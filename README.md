# Mini Project – Admin Page
노베이스가 국비학원에서 진행한 Mini Project

- HTML / CSS / JavaScript 기본 문법 학습
→ 웹 페이지를 구성하기 위한 마크업, 스타일링, 동적 기능의 기초 문법을 익힘

- 기본 구조 및 레이아웃 설계 연습
→ Header, Sidebar, Content 영역 등 Admin Page의 기본 구조를 직접 구성하며 레이아웃 이해도 향상

- 간단한 관리자(Admin) 페이지 기능 구현
→ 로그인 화면, 회원 관리 UI, 다크모드 전환 등 기초적인 관리자 페이지 기능을 JavaScript로 구현
